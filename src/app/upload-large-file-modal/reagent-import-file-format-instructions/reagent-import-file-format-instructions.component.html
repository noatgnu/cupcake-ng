Reagent Import Instructions

File Format

Supported formats: CSV, Excel (.xlsx, .xls), or Tab-delimited (.txt)

Required Columns
  • item_name: Name of the reagent
  • unit: Unit of measurement
  • quantity: Amount of reagent
Optional Columns
  • notes: Additional information
  • barcode: Barcode identifier
  • expiration_date: Expiration date in valid date format
Allowed Units

Units must be one of the following:

  • Mass: ng, ug, mg, g
  • Volume: nL, uL, mL, L
  • Molarity: uM, mM, nM, M
  • Count: ea, other
